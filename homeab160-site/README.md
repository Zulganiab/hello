# homeab160.uk — Starter Site

This is a lightweight Next.js + Tailwind starter implementing a dark, glassmorphism, neon-accented design with Framer Motion animations.

Quick start

1. Install dependencies

```bash
cd homeab160-site
npm install
```

2. Run dev server

```bash
npm run dev
```

Open http://localhost:3002

Notes

- The project includes optional Three.js and React-Three-Fiber deps for progressive enhancement; the sample uses CSS placeholders for 3D elements. Replace placeholders with a `<Canvas>` from `@react-three/fiber` for advanced 3D.
- Adjust `tailwind.config.js` content paths if you move files.
